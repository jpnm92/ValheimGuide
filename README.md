# ValheimGuide

An in-game progression guide for Valheim. Tracks where you are in the game, shows recommended gear, boss info, drop sources, crafting recipes, and objectives for each biome — all without leaving the game. [cite_start]Supports vanilla progression and optionally integrates with **Therzie's Armory** and **Therzie's Warfare** if those mods are installed[cite: 383, 384].

---

## Features

- [cite_start]**Auto-detects your current progression stage** based on which bosses you have defeated [cite: 385]
- [cite_start]**Per-stage overview** — priority materials, boss location, summon requirements, recommended gear, unlock rewards, and one-line fight strategy [cite: 385]
- [cite_start]**Build checklist** — station-by-station build objectives per biome, auto-ticked when you place the piece in-game [cite: 385]
- [cite_start]**Objectives** — crafting, gathering, and boss objectives with automatic completion detection [cite: 385]
- **On-screen quest tracker** — WoW-style collapsible overlay showing current objectives. [cite_start]**Includes live material tracking for crafting and building requirements.** [cite: 385]
- [cite_start]**Read mode** — full long-form biome guide with boss strategies, rare discoveries, and advanced tips [cite: 385]
- [cite_start]**Gear tab** — craftable armor and weapons with full recipes, station requirements, and playstyle highlights (★) [cite: 385, 386]
- [cite_start]**Drops tab** — mob entries with HP, spawn chances, resistances, drop tables, and taming info [cite: 386]
- [cite_start]**Recipes tab** — consumables and misc crafts with ingredients and station level [cite: 386]
- [cite_start]**Filters** — filter gear by damage type, armor class, and source (Vanilla / Armory / Warfare) [cite: 386]
- [cite_start]**Checkboxes** — mark off gear and recipes you have already crafted, saved per character [cite: 386]
- [cite_start]**Search bar** — filter any tab by item name [cite: 386]
- [cite_start]**Playstyle system** — choose your combat style on first launch; matching gear is highlighted in each tier [cite: 386, 387]
- [cite_start]**Spoiler control** — choose whether to see future biome content before you reach it [cite: 387]
- [cite_start]**Stage memory** — the guide remembers which stage and view mode you last had open [cite: 387]
- [cite_start]**Armory + Warfare support** — if Therzie's Armory or Warfare are installed, their items are automatically generated and grouped under the correct biome tier [cite: 387]

## Installation

### Recommended — r2modman or Thunderstore Mod Manager

1. [cite_start]Install [r2modman](https://thunderstore.io/package/ebkr/r2modman/) or the Thunderstore Mod Manager [cite: 387]
2. [cite_start]Search for **ValheimGuide** and click Install [cite: 387]
3. [cite_start]Dependencies (BepInEx, Jotunn) are installed automatically [cite: 387]
4. [cite_start]Launch Valheim through the mod manager [cite: 387]
